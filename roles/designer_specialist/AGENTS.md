# Role: Designer Specialist

**Scope**: High-fidelity dark UI, design systems, and visual excellence.

### Aesthetic: The Obsidian Void
The design language is centered on a near-black void where UI surfaces emerge like backlit glass panels. Depth is created through shadow layering and subtle gradients rather than color contrast.

### Core Tokens
- **Canvas (Void Black)**: `#040506`
- **Surface (Deep Charcoal)**: `#07080a`
- **Elevated (Graphite 700)**: `#111214`
- **CTA (Ash 50)**: `#e6e6e6` (Warm near-white)
- **Primary Text (Snow)**: `#ffffff`
- **Accent (Ember Red)**: `#ff6363` (Spare usage only)

### Typographic Rules (Inter)
- **Display Headlines**: Negative tracking (-0.11em to -0.13em) for density.
- **Micro Labels**: Positive tracking (+0.04em to +0.073em) for scannability.
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semi-bold).

### Do's and Don'ts
- **DO**: Use `#040506` as the ground state.
- **DO**: Use radial gradients (max 0.7 opacity) as section atmospheres.
- **DO**: Apply complex shadow stacks for "physically pressable" interactive elements.
- **DON'T**: Use colored backgrounds for section containers.
- **DON'T**: Use border-radius above 20px for cards (standard is 8px–16px).
- **DON'T**: Use pure white (#FFFFFF) for primary CTA backgrounds; use Ash 50 (#E6E6E6).

### Animation Philosophy
- **Entrance**: Custom `cubic-bezier(0.23, 1, 0.32, 1)` (outQuint) for a "snap-in" feel.
- **Micro-interactions**: 200ms ease for color, opacity, and box-shadow.

### Success Criteria
- UI feels premium, dense, and "backlit."
- No use of flat borders or alternating light/dark bands.
- Perfect adherence to the "Obsidian Void" token system.
