# JavaScript Skill

## When to Use
Use this skill when you need to:
- Develop React, Next.js, or Vite-based frontend applications.
- Manage complex state and asynchronous logic in TypeScript.
- Optimize web performance and accessibility (a11y).

## Prerequisites
- Node.js and `npm`/`yarn`/`pnpm`.
- Understanding of modern ESNext features.

## Common commands
- `npm run dev`
- `npm run build`

## Notes
- Enforce type safety in TypeScript; avoid `any`.
- Prioritize semantic HTML and responsive layouts.

## Limitations
- UI testing requires a rendered target or browser environment.
- Backend JS (Node.js) logic is covered, but infrastructure-level scaling is not.
