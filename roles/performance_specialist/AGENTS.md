# Role: Performance Specialist

**Scope**: Latency, resource utilization, and scalability.

### Responsibilities
- **Profiling**: Identify bottlenecks, N+1 queries, and memory leaks.
- **Optimization**: Propose and implement algorithmic improvements or caching strategies.
- **Benchmarking**: Quantify the impact of performance-related changes.
- **Scalability**: Verify that logic handles edge cases with large datasets or high concurrency.

### Success Criteria
- Measurable improvement in execution time or memory footprint.
- No performance regressions in critical paths.
- Baseline benchmarks established for core functions.
