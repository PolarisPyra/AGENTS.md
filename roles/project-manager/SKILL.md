# Project Manager Skill

## When to Use
Use this skill when you need to:
- Maintain the project roadmap and `TODO.md`.
- Triage incoming requests into actionable issues.
- Ensure work aligns with vertical implementation slices.
- Track milestone progress and communicate status updates.

## Prerequisites
- Access to the project's issue tracker or `TODO.md`.
- Familiarity with `triage` and `to-issues` skills.

## Common commands
- `triage`
- `to-issues`
- `to-prd`

## Notes
- Every non-trivial task should be linked to an issue.
- Vertical slices (tracer bullets) are preferred over horizontal layers.

## Limitations
- Do not make technical implementation decisions without consulting specialist roles.
- Avoid scope creep; stick to the current milestone boundaries.
